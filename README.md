# ORAFI Smart Contract

## Overview
This directory contains the smart contract code for the orafi Merchant Payment Gateway, deployed on the Sui blockchain.

## Purpose
- Implements core payment logic, merchant registration, wallet management, and payment status tracking on-chain.
- Ensures secure, transparent, and decentralized handling of merchant and customer transactions.

## Structure
- **contracts/**: Sui Move smart contract source files.
- **scripts/**: Deployment, testing, and interaction scripts.
- **tests/**: Unit and integration tests for contract logic.

## Usage
1. Review and modify contract code in the `contracts/` directory as needed.
2. Use provided scripts to deploy and interact with the contract on Sui testnet or mainnet.
3. Run tests to verify contract functionality and security.

## Documentation
- Refer to the main SuiPay documentation for integration guides and API references.
- Contract functions are documented in code comments for developer reference.
